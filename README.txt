###Getting and Cleaning Data - Course Project

This repository contains the run_analysis.R code and documentation files for the project for the Data Science's track course "Getting and Cleaning data", available in coursera.

It contains

  - README.md
  - CodeBook.txt : describes the variables and any transformations or work that was performed to clean up the data
  - run_analysis.R: contains all the code to perform the analyses described in the 5 steps

>Running run_analysis.R will create a folder "ProjectGetCleanData" in your home folder, it will set up this folder as the working directory, it will download and unzip the .zip file containing the data there and it will start working on the exercise.

Zip file is 
[https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip](https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip)

Finally a file named "tidyData.txt" will be created in the same folder
